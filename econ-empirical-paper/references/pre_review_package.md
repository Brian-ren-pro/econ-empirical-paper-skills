# Pre-Review Package Requirements

Use this before sending a draft to `econ-empirical-paper-reviewer`.

## Minimum Package

1. Complete manuscript draft.
2. Table/figure role map.
3. Exhibit map with display numbering and source-file mapping.
4. Result-bundle summary.
5. Literature/mechanism summary.
6. Unresolved issues list.

## Exhibit Map Template

| Display Label | Source File | Section | Role | Notes |
|---|---|---|---|---|
| Table 3 | tables/Table3_main_new_loan_contracting.csv | 4.2 | main | core term-loan result |

## Variant Handling

Projects often have:

- source manuscript without tables;
- inline-exhibits manuscript;
- builder script that maps source files to display table numbers;
- DOCX/PDF rendered outputs.

Do not assume source file names match displayed table numbers. If the exhibit map is missing, generate it from the builder script or result bundle before full review.

## Review Boundary

If the exhibit map is missing:

- full evidence review should pause;
- prose-only review is allowed only if the user explicitly requests it.
